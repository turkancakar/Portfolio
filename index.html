E-Commerce Platform
Proje Açıklaması

Modern bir e-ticaret uygulaması. Backend Spring Boot (JWT, JPA, Swagger, Elasticsearch) ile, frontend React + Vite + Tailwind ile geliştirilmiştir. Proje; kullanıcı/satıcı yönetimi, ürünler, kategoriler, arama, favoriler, sepet/sipariş, yorumlar ve satıcı paneli gibi modüller içerir.

Özellikler
  • JWT tabanlı kimlik doğrulama ve rol tabanlı erişim (ADMIN, SELLER, USER)
  • Ürün ve kategori yönetimi ile filtreleme
  • Elasticsearch ile tam metin arama ve arama önerileri
  • Sepet yönetimi ve sipariş oluşturma/takip
  • Favoriler ve bildirim sistemi
  • Ürün değerlendirme ve yorumlar
  • Satıcı paneli (stok, sipariş, ürün yönetimi)
  • Admin paneli ve Swagger/OpenAPI dokümantasyonu
  • Responsive tasarım ve modern UI
Kullanılan Teknolojiler
React 19 Spring Boot 3.5.3 PostgreSQL Elasticsearch JWT Tailwind CSS Vite Swagger

Weather Forecast
Proje Açıklaması

Tek sayfalık bir hava durumu uygulaması. OpenWeather API'sini kullanarak gerçek zamanlı hava durumu bilgilerini gösterir. Varsayılan olarak İstanbul'un hava durumu gösterilir ve kullanıcılar başka şehirler için arama yapabilir.

Özellikler
  • Şehir bazlı hava durumu sorgulama
  • Max-min sıcaklık ve hava durumu açıklaması
  • Modern ve responsive tasarım
  • Türkçe/İngilizce dil desteği
  • Hata yönetimi ve kullanıcı uyarıları
Kullanılan Teknolojiler
HTML5 CSS3 JavaScript (ES6+) Node.js Express.js OpenWeather API
